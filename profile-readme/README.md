## Featured Projects

### [GlowUp OS](https://github.com/learnfromothers32-cell/glowup-backend-)
> The operating system for beauty professionals.

Full-stack marketplace platform with real-time booking, live streaming, AI hairstyle generation, POS, and integrated payments.

**Stack:** React 19 · TypeScript · Express · MongoDB · Redis · Socket.IO · Paystack · HuggingFace AI · Docker

**Highlights:**
- 33 Mongoose models, 28 controllers, 164 passing tests
- Provider-agnostic payment architecture (Paystack + ready for Stripe/MoMo/M-Pesa)
- Real-time live streaming with WebRTC signaling and Socket.IO chat
- AI-powered hairstyle previews via HuggingFace FLUX model
- Full POS system with barcode scanning and real-time inventory
- 10-language internationalization
- Microsoft Clarity analytics integration
- Production-deployed on Render + Vercel

### [Own-Shop](https://github.com/learnfromothers32-cell/Own-Shop)
> Full-stack e-commerce platform for Ghanaian businesses.

MTN Mobile Money + Stripe card payments, real-time admin notifications, and analytics dashboard.

**Stack:** React 19 · Express 5 · MongoDB · Stripe · MTN MoMo · Socket.IO · Cloudinary · Brevo

**Highlights:**
- 3 payment methods (Stripe, MTN MoMo, Cash on Delivery)
- Real-time admin notifications via Socket.IO
- Admin dashboard with 7-day sales trend + payment breakdown
- MTN MoMo sandbox with test phone numbers
- Brevo API email service with HTML templates
- Push notifications via web-push
- Google Analytics integration

## How I Work

```
Architecture first  →  Type everything  →  Test the hard parts  →  Deploy with confidence
```

**My approach to payments:**
```
Interface → Factory → Provider Implementation → Webhook Handler → Atomic Session
```
Every payment write is atomic. Every webhook is verified. Every provider is swappable.

**My approach to real-time:**
```
Socket.IO namespaces → Redis adapter (horizontal scale) → Room-based broadcasting → WebRTC signaling
```
